Your Browser Is Not Supported

We’re sorry but POPVOX does not support your web browser. If you are using Microsoft Windows XP or later, we recommend that you download and install Google Chrome or update to the latest version of Internet Explorer.

You are using the Internet Explorer 6 web browser, which was released in 2001. That’s nearly a lifetime ago in Internet years. Because the point of POPVOX is to use technology to innovate our civic lives, we would not be able to fulfill our mission by including support for web browsing software a decade old. Major websites began dropping support for Internet Explorer 6 starting in 2010.

If you are using Windows 98, 2000, or ME, you will not be able to use POPVOX on this computer.

We recognize that upgrading is not an option for everyone. In that case, we hope that you will continue to communicate with your Members of Congress through their websites directly. You can also try to access POPVOX at your local library.

Thank you for your understanding.
