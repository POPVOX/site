Using In Microsoft Outlook 2007

From this Microsoft page:

  • Right-click either this House or Senate link and choose Copy Shortcut, Copy Link Address, or Copy Link Location.
  • In Outlook, on the Tools menu, click Account Settings.
  • On the Internet Calendars tab, click New.
  • Paste the Internet Web address you copied by pressing CTRL+V.
  • Click Add.
  • In the Folder Name box, type the name of the calendar as you want it to appear in Outlook, and then click OK.
